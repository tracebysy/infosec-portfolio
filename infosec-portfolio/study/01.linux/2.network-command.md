# 네트워크 명령어
# 개념
리눅스에서 네트워크 상태를 확인하고 진단하기 위한 명령어

---

## 주요 명령어
1. ip / ifconfig
(1) #ip addr
(2) #ifconfig : 내 컴퓨터의 정보 확인 -> vi editor를 통해 수정 가능
2. ping
ex) #ping 8.8.8.8
3. netstat : 네트워크 상태를 확인 ( 현재 열려있는 포트 / 연결 상태 / 라우팅 정보 등)
(1) #netstat -an -> 모든 연결을 숫자로 표시

---

>!> 중요 포인트
- IP 확인
- 연결 상태 확인
- 문제 진단 가능
