# 개념
리눅스 기본 텍스트(문서) 편집기
즉, 텍스트 파일이나 설정 파일 내용을 수정하기 위한 문서 편집기 이다.

---

## 기본 모드
1. 명령 모드
- vi 편집기의 기본 모드로 다른 모드로 전환할 수 있다.
- 명령 모드에서는 간단한 검색 및 커서 이동, 복사/붙여넣기 등의 편집을 수행 할 수 있다.
  
2. (편집)입력 모드
- 문서에 텍스트를 입력/수정/삭제 등으로 내용을 작성할 수 있다.
  
3. 실행 모드
- vi 편집기를 제어하는 명령어를 실행 한다. -> 문서 편집을 제외한 나머지 지능을 구현한다.
- 치환/저장/파일 읽기/환경 설정 등

---

## 모드 변 명령어
- i → 명령 모드 -> 입력 모드  
- esc → 입력/실행 모드 -> 명령 모드  

### 명령 모드의 주요 명령어
- u → 되돌리기(실행 취소)
- gg → 문서의 가장 첫 번째 행으로 이동
- G → 문서의 가장 마지막 행으로 이동
- yy → 현재 위치한 행을 복사(임시 저장)
- p → 복사한 내용을 현재 위치한 행의 아래쪽으로 붙여넣기
- dd → 현재 위치한 행 잘라내기
- 
### 실행 모드의 주요 명령어
1. 검색기능 활성화
- ' ? ' → 커서를 기준으로 위의 내용 중 검색
- ' / ' → 커서를 기준으로 아래의 내용 중 검삭
- n → 검색 결과 중 다음으로 넘김
- nohl → 검색기능의 하이라이트 없애기
3. %s /문자1/문자2 /g → 문자 1을 문자 2로 치환
4. r! <명령어> : shell에서 실행 된 명령의 결과를 vi로 출력
5. 숫자 → 원하는 줄로 이동
6. set nu → 행 번호 표시
- set nonu : 행 번호 제거
7. w → 문서 저장
- w! → 문서 강제 저장
8. q → 종료(저장x)
q! → 강제 종료(저장x)
