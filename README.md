# infosec-portfolio

## 📌 소개
비전공자에서 정보보안 분야로 전향하며,
웹 취약점 분석 및 공격/방어 원리를 이해하기 위해 DVWA 기반 실습을 진행한 포트폴리오입니다.
단순한 공격 수행이 아니라,
각 취약점이 발생하는 원인과 대응 방법까지 함께 학습하고 정리했습니다.
각 항목은 개념 → 실습 → 보고서(결과 및 대응 방안)의 흐름으로 정리되어 있습니다.
