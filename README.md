# infosec-portfolio

## 소개
비전공자에서 정보보안 분야로 전향하며,  
웹 취약점 분석 및 공격/방어 원리를 이해하기 위해  
DVWA 기반 실습을 수행하고 정리한 포트폴리오입니다.

단순한 공격 수행이 아닌,  
"취약점이 발생하는 원리"와 "대응 방법"까지 함께 학습했습니다.

## 기술 스택
- Linux (기본 명령어 및 환경 구성)
- Network (TCP/IP, HTTP 구조 이해)
- Web Hacking
  - SQL Injection
  - XSS
  - Command Injection

- ## 목표
보안관제 및 침해대응 직무를 목표로,  
실무에서 활용 가능한 공격 흐름 이해와 로그 분석 역량을 키우고 있습니다.
